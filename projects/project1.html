Saturn Offshore Drilling Rig SCR Network and Rig-Floor Automation Modernization Project

Client: EDC Romfor Ltd. – Saturn Rig | Location: Saturn Offshore Drilling Rig, Caspian Sea | Commissioning Date: June 16, 2026 | Completed
Project Overview

Watchdog Automation completed a multi-package automation modernization and commissioning project onboard the Saturn Offshore Drilling Rig. The project consolidated several critical control and monitoring upgrades under one integrated modernization scope, covering the SCR Main PLC network, operator interface systems, Mud Pump current monitoring, Hydraulic Power Unit control, and Doghouse Remote I/O infrastructure.

The works were performed to address hardware obsolescence, limited spare parts availability, reduced legacy-system support, and increasing maintenance difficulty across the rig’s existing automation architecture. By migrating key systems to modern Siemens automation platforms, the project improved system diagnostics, reliability, maintainability, operator visibility, and long-term supportability for drilling operations.

The modernization preserved existing operational functions where required while introducing upgraded PLC, HMI, Remote I/O, signal acquisition, and monitoring capabilities. This allowed the rig’s control systems to remain operationally familiar to personnel while improving technical performance and future expansion readiness.

Integrated Commissioning Scope

The overall project consisted of five major commissioning packages, each focused on a specific automation subsystem within the rig’s SCR network and rig-floor control architecture.

SCR Main PLC migration from Siemens S7-300 to Siemens S7-1500
Siemens MTP1200 Unified Comfort HMI installation and migration
Mud Pump armature current monitoring system implementation
Hydraulic Power Unit PLC control system upgrade
Doghouse Remote I/O migration from Siemens ET200B to ET200SP
Profibus DP and Profinet communication verification
Digital and analog I/O testing, calibration, and validation
Final commissioning, operational testing, and client acceptance
01
Purchase Order No. S26-0029

SCR Main PLC Controller Upgrade

The obsolete Siemens S7-300 CPU 315-2DP was migrated to a Siemens SIMATIC S7-1500 CPU 1513 platform with Profibus DP communication capability. The upgraded PLC serves as the central controller for the rig’s SCR network and coordinates critical drilling equipment including generators, mud pumps, drawworks, rotary table, remote I/O stations, and operator interfaces.

The migration was completed while maintaining compatibility with the existing Profibus DP field network and operational control philosophy. Software and communication mapping adjustments were implemented to ensure that field devices, HMI displays, generator status signals, mud pump monitoring, and equipment interlocks functioned correctly under the upgraded PLC architecture.

Migration from Siemens S7-300 CPU 315-2DP to Siemens S7-1500 CPU 1513
Integration and verification of 22 Profibus DP slave devices
Mud Pump, Generator, Drawworks, and Rotary Table operational verification
Doghouse HMI communication and visualization compatibility adjustments
Improved diagnostics, communication stability, and maintainability
02
Purchase Order No. S26-0067

Siemens MTP1200 Unified Comfort HMI Installation

Two Siemens MTP1200 Unified Comfort Panels were supplied, installed, tested, and commissioned for the SCR Network Monitoring and Control System. The HMI upgrade replaced obsolete visualization hardware and provided improved operator visibility, alarm monitoring, equipment diagnostics, and remote monitoring capability.

The HMI application was migrated and enhanced from the legacy Siemens MP277 platform to the Siemens Unified environment. The new HMI stations provide real-time monitoring of generators, mud pumps, drawworks, rotary table, SCR alarms, communication diagnostics, and equipment health information.

Installation of two Siemens MTP1200 Unified Comfort Panels
Migration and redevelopment of legacy MP277 HMI screens
Profinet communication with the upgraded Siemens S7-1500 SCR Main PLC
Primary operator interface installed in the SCR control area
Remote monitoring and diagnostic station installed for maintenance personnel
03
Purchase Order No. S26-0053

Mud Pumps Current Monitoring System

A real-time Mud Pump armature current monitoring system was installed for Mud Pump No. 1, Mud Pump No. 2, and Mud Pump No. 3. Each mud pump was equipped with monitoring for Motor A and Motor B using LEM HOOP 2000-SB current transducers connected to Siemens PLC analog inputs.

The system converts transducer voltage signals into engineering values in amperes and displays the resulting values through the upgraded Siemens MTP1200 Unified Comfort HMI. This provides operations and maintenance personnel with visibility of current sharing, motor loading, and SCR drive performance.

Six LEM HOOP 2000-SB current transducers installed
Analog signal range configured from 0–4 VDC equivalent to 0–2000 A
All six monitoring channels calibrated and verified
Real-time HMI visualization of Mud Pump Motor A and Motor B currents
Improved troubleshooting visibility and predictive maintenance support
04
Purchase Order No. S26-0077

Hydraulic Power Unit PLC Control System Upgrade

The obsolete EATON D50 PLC-based control system for the Hydraulic Power Unit was replaced with a Siemens SIMATIC S7-1200 PLC platform. The HPU serves as an auxiliary hydraulic system supporting the Drawworks braking system and required a reliable, maintainable, and supportable control platform.

The upgraded system maintained local panel operation and Doghouse remote operation while improving diagnostics and troubleshooting visibility. Digital inputs, digital outputs, relay interfaces, power supply performance, alarm signals, and operational control functions were tested and verified.

Replacement of obsolete EATON D50 PLC with Siemens S7-1200 PLC
Verification of 9 digital inputs and 9 digital outputs
Local start/stop and Doghouse remote control verification
Low oil level, high temperature, and motor trip alarm testing
Improved PLC diagnostics and documented program troubleshooting support
05
Purchase Order No. S26-0114

Doghouse Remote I/O Migration from Siemens ET200B to ET200SP

The Doghouse Remote I/O system was migrated from obsolete Siemens ET200B hardware to the modern Siemens ET200SP distributed I/O platform. The upgrade was performed to address hardware obsolescence, improve spare parts availability, enhance diagnostics, and provide a scalable platform for future expansion.

The migration preserved the existing PLC I/O addressing to maintain software compatibility while improving maintainability and diagnostic capability. Digital inputs, digital outputs, analog inputs, analog outputs, Profibus DP communication, and additional drilling parameter monitoring signals were tested and validated.

Migration from Siemens ET200B Remote I/O to Siemens ET200SP distributed I/O
64 digital inputs and 32 digital outputs tested and verified
24 analog input channels calibrated and validated
3 analog output channels verified
Additional drilling parameters integrated into the SCR PLC network
Profibus DP communication integrity confirmed stable
Commissioning Results

All major commissioning packages were completed successfully. The upgraded systems demonstrated stable communication, correct field signal response, verified PLC and HMI functionality, reliable alarm handling, and proper operational behavior under tested rig conditions.

The final commissioning status confirmed that the systems were accepted and recommended for operational service. The project strengthened the rig’s automation infrastructure by improving diagnostics, maintainability, operator visibility, spare parts support, and readiness for future modernization phases.

  • SCR Main PLC upgraded to Siemens S7-1500 platform and verified operational
  • Siemens MTP1200 Unified Comfort HMIs installed and commissioned
  • Mud Pump current monitoring channels calibrated and displayed on HMI
  • HPU PLC control system upgraded from EATON D50 to Siemens S7-1200
  • Doghouse Remote I/O migrated from ET200B to ET200SP
  • Project closed successfully and accepted for continuous operation
Recommended Future Enhancements

Following the successful commissioning of the upgraded systems, several future enhancements were recommended to continue improving the rig’s automation architecture and reduce dependence on legacy control platforms.

Migrate remaining Profibus DP communication networks to Profinet where practical
Replace the remaining obsolete Doghouse MP277 HMI with a Siemens Unified Comfort Panel
Integrate upgraded HPU and Mud Pump PLCs into the centralized Siemens automation network
Maintain strategic spare parts for PLC, HMI, Remote I/O, power supply, and communication hardware
Consider future SCR drive modernization using modern Siemens DC drive technology
Engineering Value Delivered

This project demonstrates Watchdog Automation’s capability to execute complex offshore automation modernization works involving legacy system migration, Siemens PLC engineering, HMI integration, Remote I/O replacement, instrumentation calibration, communication troubleshooting, and full commissioning support.

By combining multiple modernization scopes into a coordinated execution program, the project delivered a more reliable, maintainable, and future-ready automation platform for Saturn Rig’s SCR network and rig-floor control systems.
